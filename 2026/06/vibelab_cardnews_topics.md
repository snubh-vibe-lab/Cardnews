# Vibe Lab 카드뉴스 토픽 기획안 9개

대상: 분당서울대병원 중견교수 AI 도구 학습/실습 모임(Vibe Lab)

목적: 교수님들이 매월 실습에서 바로 따라 해보고, 진료·연구·교육·행정 업무에 안전하게 적용할 수 있는 카드뉴스 주제를 정리합니다. 환자 식별정보, 내부 기밀, 미공개 연구자료 등 민감정보는 포함하지 않습니다.

## 1. 임상교수용 AI 업무자동화

- 핵심 메시지: 반복되는 문서 정리, 회의록, 할 일 목록을 AI로 줄일 수 있습니다.
- 카드 구성: 문제 상황 → 자동화 가능한 업무 → 프롬프트 예시 → 검토/책임 원칙
- 실습 아이디어: 회의 메모를 입력해 요약, 결정사항, 담당자별 To-do로 변환하기

## 2. 논문/문헌요약 워크플로

- 핵심 메시지: PubMed 논문, 진료지침, 리뷰 문헌을 빠르게 선별하고 구조화할 수 있습니다.
- 카드 구성: 검색식 만들기 → 초록 요약 → PICO/근거수준 정리 → 원문 확인 체크리스트
- 실습 아이디어: 공개 초록을 바탕으로 연구 질문, 주요 결과, 한계점 표준 양식 만들기

## 3. 진료/행정 이메일·일정 자동화

- 핵심 메시지: 정중한 이메일 초안, 안내문, 일정 조율 문구를 빠르게 만들 수 있습니다.
- 카드 구성: 상황별 이메일 템플릿 → 일정 조율 → 회신 톤 조정 → 발송 전 검토
- 실습 아이디어: 학회/연구미팅/부서회의 안내 메일 초안을 만들고 교수님 스타일로 다듬기

## 4. 연구 데이터/통계·코딩 보조

- 핵심 메시지: AI는 Python/R 코드 작성, 오류 설명, 분석 계획 정리에 유용한 보조자입니다.
- 카드 구성: 분석 질문 정의 → 코드 생성 → 오류 해결 → 결과 해석 시 주의점
- 실습 아이디어: 비식별 예제 데이터로 기술통계, 시각화, 간단한 회귀분석 코드 생성하기

## 5. 교육자료·강의 슬라이드 생성

- 핵심 메시지: 전공의·학생 교육용 강의 개요, 퀴즈, 슬라이드 초안을 빠르게 만들 수 있습니다.
- 카드 구성: 학습목표 → 강의 흐름 → 사례 기반 질문 → 평가 문항
- 실습 아이디어: 30분 미니강의 목차와 슬라이드별 핵심 문장을 생성하기

## 6. 의료 AI 윤리·보안·개인정보

- 핵심 메시지: 병원 업무에서 AI를 쓸 때는 개인정보, 보안, 책임소재를 먼저 점검해야 합니다.
- 카드 구성: 입력 금지 정보 → 비식별화 원칙 → 도구별 보안 확인 → 최종 판단은 전문가
- 실습 아이디어: 안전한 프롬프트와 위험한 프롬프트를 비교해 체크리스트 만들기

## 7. GitHub 협업/버전관리

- 핵심 메시지: GitHub는 실습 자료, 프롬프트, 코드, 카드뉴스를 안전하게 함께 개선하는 공간입니다.
- 카드 구성: 저장소 구조 → 브랜치/커밋 → Pull Request → 리뷰 문화
- 실습 아이디어: 카드뉴스 토픽 파일을 수정하고 커밋 메시지 작성 규칙을 연습하기

## 8. 로컬 LLM·프롬프트 설계

- 핵심 메시지: 로컬 LLM과 좋은 프롬프트 설계는 안전한 실습과 반복 가능한 결과를 돕습니다.
- 카드 구성: 로컬/클라우드 차이 → 역할·목표·제약 조건 → 예시 제공 → 결과 검증
- 실습 아이디어: 동일한 요청을 나쁜 프롬프트와 좋은 프롬프트로 비교해 결과 차이를 확인하기

## 9. 실제 병원 업무 적용 사례

- 핵심 메시지: 작은 자동화 사례를 모아 부서별 업무 개선으로 확장할 수 있습니다.
- 카드 구성: 적용 전 업무흐름 → AI 적용 지점 → 시간 절감/품질 개선 → 확산 조건
- 실습 아이디어: 교수님별로 한 가지 반복 업무를 선정해 AI 적용 가능성과 위험도를 점검하기

## 제작 메모

- 톤앤매너: 교수님 대상의 실용적이고 차분한 문체를 사용합니다.
- 분량: 주제별 4~5장 카드 구성을 기본으로 합니다.
- 금지: 환자 식별정보, 내부 계정/시스템 정보, 미공개 연구자료, 실제 진료기록 예시는 사용하지 않습니다.
- 권장: 공개 자료, 가상 사례, 비식별 예제 데이터, 도구 사용 전 검토 체크리스트를 사용합니다.
