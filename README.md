## 신규 PR: Vibe Coding & Agentic AI Tips

월례모임 발표 주제형 안은 폐기하고, 메신저에서 바로 읽히는 단순 바이브 코딩·agentic AI 실전 팁 카드뉴스로 재작성했습니다.

위치: `2026/06/vibe-coding-agentic-tips/`

포함 토픽: 각 4장, 총 28장

1. 고칠 파일부터 찍어주기
2. 요구사항은 3줄로 쪼개기
3. 수정 전에 읽기만 시키기
4. Agent에게 권한 경계 주기
5. “안 돼” 대신 증거를 주기
6. 결과 보고 형식을 정해두기
7. 자주 쓰는 지시는 템플릿으로 저장

## 운영 메모

- 대상 독자는 “AI에 관심은 있지만 개발자가 아닌 중견 임상의”로 둡니다.
- 카드뉴스는 발표자료가 아니라, 바로 따라 할 수 있는 짧은 실전 팁 톤을 유지합니다.
- 기존 이미지 묶음과 중복되지 않도록 구체적 사용 습관·작업 지시법 중심으로 확장합니다.
- 민감정보, 환자정보, 계정정보는 카드뉴스 예시에도 포함하지 않습니다.
